Machine Learning Project

Customer Churn Prediction

This project focuses on predicting customer churn using various machine learning algorithms. The dataset includes customer demographics, account information, and service usage patterns. The goal is to identify customers who are likely to leave the service so that proactive retention strategies can be implemented.

Technologies Used:

  • Python
  • Scikit-learn
  • Pandas & NumPy
  • Jupyter Notebooks
  • Homelab Deployment

Project Highlights:

  • Data preprocessing and feature engineering to improve model accuracy.
  • Comparison of multiple algorithms including Logistic Regression, Decision Trees, and Random Forests.
  • Model evaluation using accuracy, precision, recall, and F1-score metrics.
  • Deployment of the best-performing model on a personal Homelab server for real-time predictions.
